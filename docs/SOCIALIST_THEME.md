# Co-opoly Deal: Socialist Theme Translation Key

This document outlines the alternate "Socialist/Marxist" theme for Co-opoly Deal, replacing the classic capitalist Monopoly terms with state-controlled, collectivistic equivalents.

## Core Metadata

- **Currency:** Rubles (symbol: **₽**)
- **Property Sets:** Collectives
- **Action Cards:** Directives
- **Money Pile:** Communal Ledger
- **Houses/Hotels:** Outposts / Monuments
- **Tone:** Authoritarian, industrial, bureaucratic, and collectivistic.

## 4. Branding Guidelines & UX

- **Naming Convention:** Short, punchy, and utilitarian.
- **Terminology:** 
  - Players are referred to as **Comrades**.
  - Rent is referred to as **Levy**.
- **Victory Condition:** *"Be the first Comrade to secure three full Collectives to stabilize the Revolution."*
- **Flavor Text Logic:** Use phrases like *"For the glory of the worker,"* *"Non-negotiable contribution,"* and *"By order of the Politburo."*
- **Dialog Prompts:** Add randomized quirky sayings based on the opponent's type (e.g., Capitalists vs. Socialists). Example: *"Those greedy capitalists are stealing our hard earned wages again."*
